Detailed Description

template<typename T, size_t Dim = 3>
class Kdtree< T, Dim >

template class for implementing kd-tree.

Note:
You can integrate with any own vector types, but it must have following two methods:
  • T::operator[](int);
  • T::T(const T& d);
  • T::copy( const T& d) copying entity of d.//T::operator=( const T& d )
